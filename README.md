# Delia's Art Portfolio

A beautiful, immersive art portfolio website featuring original paintings and mixed media artwork by Delia.

## ✨ Features

### 🎨 Interactive Story Experience
- **Mobile**: Cinematic video story with interactive pause points (tap, hold, swipe)
- **Desktop**: Elegant web intro with smooth transitions
- Seamless transition to gallery

### 🖼️ Gallery & Collection
- Responsive grid layout with video previews
- Smooth hover effects with subtle animations
- Custom brush cursor for artistic feel
- Optimized video loading from Cloudinary CDN

### 📱 Product Pages
- Full-screen artwork display with detailed information
- **Share functionality**: Facebook, Twitter, WhatsApp, Copy Link
- **Related artworks**: Random suggestions to explore more pieces
- Smooth navigation and transitions

### 👩‍🎨 About Page
- Artist story and creative process
- Direct Instagram integration (@delia.sart)
- Beautiful typography and layout

### 🔍 SEO Optimized
- Complete meta tags (Open Graph, Twitter Cards)
- Dynamic page titles and descriptions per artwork
- Canonical URLs and proper indexing
- Optimized for social media sharing

### 🎯 User Experience
- Custom brush cursor throughout the site
- Smooth animations and transitions
- Mobile-first responsive design
- Accessibility considerations

## 🚀 Tech Stack

- **React** + **TypeScript**
- **React Router** for navigation
- **Vite** for fast development
- **Cloudinary** for video CDN
- Custom CSS with modern features

## 📦 Development

```bash
npm install
npm run dev
```

## 🏗️ Build

```bash
npm run build
npm run preview
```

## 🌐 Deploy on Vercel

- **Framework Preset**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **SPA routing**: Includes `vercel.json` with rewrite to `index.html` for routes:
  - `/collection`
  - `/products/:id`
  - `/about`
